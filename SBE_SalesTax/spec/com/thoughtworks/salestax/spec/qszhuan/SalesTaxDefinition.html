Basic Tax
  • Basic sales tax is 10% which is applicable on all goods, except books, food, and medical products that are exempt.
    So, given the next list, the result should be as expected:
    • Good Type Exempt for basic tax Tax Rate
    Book true 0
    Food true 0
    Medical true 0
    Others false 10%
    Import Tax
  • Import duty is an additional sales tax applicable on all imported goods at a rate of 5%, with no exemptions.
    • Goods name Is imported Import duty
    1 imported box of chocolates true 5%
    1 chocolate bar false 0
